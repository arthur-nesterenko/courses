### –ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É–µ–º state

–ü–µ—Ä–µ–¥ —Ç–µ–º, –∫–∞–∫ –≤—ã—Å—Ç–∞–≤–ª—è—Ç—å state, –Ω–∞–º –Ω—É–∂–Ω–æ –≤—ã–∑–≤–∞—Ç—å super() –≤ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–µ. –≠—Ç–æ –ø–æ—Ç–æ–º—É, —á—Ç–æ this ‚Äî –Ω–µ–∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω –ø–µ—Ä–µ–¥ –≤—ã–∑–æ–≤–æ–º super() .

  constructor(props) {

      count: 0,
    };

      count: prevState.count + 1,
    }));
  };

        {this.state.count}

  }
}
